Profile completion flow (first name, last name, phone, address, DOB)
Referral code support during signup (optional field)

Splash screen animation on first visit (mobile)
Hero section with promotional banners

Admin-curated video gallery
Featured celebrities grid
"How It Works" section (3-step explainer)
Business section, AI Music section, Referral section
Video ads banner

Browse all active celebrities (/celebrities)

Celebrity profile page (/celebrity/:id) with:
Bio, stats (followers, bookings completed, response time)
Media gallery (photos/videos)

Services & pricing
Reviews tab
"Book Now" button per service
Subscription option (₹ yearly/monthly — required before booking some services)
📹 BOOKING A CELEBRITY SERVICE
Multi-step booking flow (/booking/:celebrityId):
Request Details — occasion, recipient name, message, special instructions
Delivery Type — standard or express
Creative Guidelines — style preferences
Script Builder — write or AI-generate script
Review — summary of everything
Checkout — payment via Wallet or Razorpay
Service types: personalized_video, birthday_wish, business_shoutout, ugc_ad, voice_call, live_event
Plan-based discounts (Silver 5%, Gold 10%, Platinum 15%)
Booking creates record in bookings table, then celebrity fulfills it
💰 WALLET SYSTEM
View current balance

Transaction history with filters (all, deposits, deductions, refunds)
Real-time balance updates via Supabase realtime subscriptions
Used for: booking payments, music generation, photo wishes, greeting cards, etc.

Sound Effects — describe a sound, AI generates it
Audio Isolation — upload track, isolate vocals/instruments
AI Music — generate full songs with lyrics or instrumental (MusicAPI/Suno)
Voice Cloning — upload voice sample, create custom voice
Image + Music Video — upload image, generate music, create video
Each feature costs wallet credits (configurable per feature)
Generation history with playback and download
